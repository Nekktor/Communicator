# Описание классов проекта
## Message - класс сообщения
 * Параметры:
   - type
   - content
   - replyIdMes
   - userId
   - timeChanges
 * Методы:
   - change; получает: content
   - delete
## User - класс пользователя
 * Параметры:
   - idUser
   - username
   - avatarUrl (необязательная по началу функция)
   - status
   - lastTimeOnline
 * Методы:
   - change;
## Chat - класс одного чата
 * Параметры:
   - idChat
   - name
   - avatarUrl (необязательная по началу функция)
   - participants (id)
   - timeCreating (необязательная по началу функция)
 * Методы:
   - Создать
   - Удалить
   - Добавить сообщение (связь с Message)
   - Изменить (name или avatarUrl)
