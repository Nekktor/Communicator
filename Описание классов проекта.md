# Описание классов проекта
## Message - класс сообщения
 * Параметры:
   - type
   - content
   - replyIdMes
   - userId
   - timeChanges
 * Методы:
   - change; получает: content
   - delete
## User - класс пользователя
 * Параметры:
   - idUser
   - username
   - avatarUrl (необязательная по началу функция)
   - status
   - lastTimeOnline
 * Методы:
   - change
## Chat - класс одного чата
 * Параметры:
   - idChat
   - name
   - avatarUrl (необязательная по началу функция)
   - participants (id)
   - timeCreating (необязательная по началу функция)
 * Методы:
   - Создать
   - Удалить
   - Добавить сообщение (связь с Message)
   - Изменить (name или avatarUrl)
## Connection - класс подключения сервера
 * Функционал: поддерживает связь с сервером, занимается отправкой и получением данных
## SynqQueue - класс очереди операций, отправляемых на сервер
 * Функционал: получает операцию на вход, создает из них очередь по важности (отправка сообщения > отправка реакции)
## Notification - класс уведомления (необязательная по началу функция)
 * Параметры:
   - idNot
   - userId (получатель)
   - type
   - data (информация уведомления)
   - isRead
   - timeCreating
