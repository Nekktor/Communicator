# Вспомогательные классы со стороны UI
  ## User (класс пользователя)
  * В единственном экземпляре этого класса будет хранится информация о пользователе, который пользуется приложением сейчас.
      * Параметры
        - name (str) Отображаемое имя пользователя
        - username (str) Уникальное имя пользователя
      * Методы
        - registration_check(username) Проверяет уникальность имени пользователя, возвращает bool
        - add_user(name, username) Принимает на вход данные о пользователе из полей для ввода и загружает их в базу данных, возвращает ничего
        - login_check(username) Проверяет наличие имени пользователя в БД, возвращает bool
        - get_user_data(username) Получает данные пользователя по имени пользователя в виде словаря ('name':'Юра', 'username': ...)
