# Цикл

1. Пользователь открыл впервые приложение, ему выдался уникальный id для подключения к серверу(чтобы сервер мог различать пользователей)
2. Сервер добавил id в список всех пользователей, список онлайн-пользователей
3. Пользователь зарегистрировался, отправил запрос на сервер, сервер отправил в БД, БД проверила возможно ли создать пользователя (не занят ли ник и тд), сервер вернул "OK" или "Error". Где -то на устройстве сохранился id пользователя для подключения к серверу(id хранить на устройстве пользователя и в БД)
4. Приложение создало задачу получить список чатов, сервер выполняя задачу, отправляет список чатов(в формате json-строки) и приложение эти данные открывает в UI (пока чатов нет)
5. Пользователь хочет создать группу, выбирает пользователей, например, по номеру(@nickname, или еще что-нибудь), создает задачу передавая пользователей (предлагаю сначала добавление в группу только по @nickname). Сервер отправляет запрос в БД, она создает новую группу, возращает "OK". Если "OK", то приложение снова вызывает задачу обновления чатов, но получает список с новым чатом, также вызывает задачу оповестить пользователей о добавлении пользователей в группу(передает @nickname). Когда пользователь получил уведомлении о добавлении в группу, задача обновлении чатов снова вызывается.

6. Пользователь хочет отправить сообщение, пишет его. Отправляет задачу на сервер (передав id чата(он ведь будет у чата?)). Сервер передает сообщение в БД. БД создало сообщение и вернуло новую версию чата. Сервер вернул обновленный чат. Создалась задача об уведомлении пользователей в чате, пользователи получив уведомление обновляют один чат(или сразу все, но лучше думаю сделать задачу, куда можно передать id чатов для обновления или если не передавать, то обновятся все чаты)
7. Остальные действия по аналогии (реакции и тд)
8. Когда пользоватлеь просмотрел сообщение, он отправляет новое сообщение (где к @nickname пользователей, которые просмотрели добавляется его @nickname) и выполняются все предыдущие шаги (уведомление пользователей, обновление чатов и тд)
