# Database/models/ - Классы с шаблонаби таблиц

### класс Users (таблица users):
Инфо обо всех пользователях
  * id
  * name
  * username
  * lastname
  * aavatar_url
  * date_created
  * last_time_online
  * phone
  * email
   
### класс Chats (таблица chats):
Инфо обо всех чатов (схоже по смыслу с таблицей users)
  * id
  * name
  * avatar_url
  * date_created
  
### класс Messages (таблица messages):
Инфо обо всех сообщениях
  * id
  * type (может принимать "text" или "file")
  * text
  * file_id
  * creation_date_time
  * chat_id
  * user_id
    
### класс Participants (таблица participants):
Инфо обо всех пользователях как "участниках чатов"
  * chat_id
  * user_id
  * joined_at
  * role
