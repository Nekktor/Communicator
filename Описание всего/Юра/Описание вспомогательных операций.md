# Вспомогательные операции
  ## Регистрация (класс App в main.py) register_user()
  * Метод класса приложения, обрабатывает нажатие кнопки подтверждения регистрации на sign_up_page
  * Производимые действия
    - Считать данные для регистрации из name_entry и username_entry, которые находятся в sign_up_page
    - Создать c помощью полученных данных экземпляр класса User и записать в переменную App.user
    - Проверить с помощью registration_check данные для регистрации
    - Если проверка прошла успешно, установить значения переменных в классе User на данные пользователя и перейти на chatting_page
    - Если проверка не удалась, сбросить значение переменной App.user на None и показать пользователю сообщение об ошибке
  ## Авторизация (класс App в main.py) login_user()
  * Метод класса приложения, обрабатывает нажатие кнопки подтверждения авторизации на sign_up_page
  * Проиводимые действия
    - Считать данные из username_entry, который находится в sign_up_page
    - Создать c помощью полученных данных экземпляр класса User с пустым параметром name ('') и записать в переменную App.user
    - Проверить с помощью login_check данные для регистрации
    - Если проверка прошла успешно, получить данные пользователя с помощью get_user_data, установить значения переменных экземпляра класса User и перейти на chatting_page
    - Если проверка не удалась, сбросить значение переменной App.user на None и показать пользователю сообщение об ошибке
  ## Получить название текущей страницы (класс App в main.py) get_current_page()
  * Возвращает строку с названием текущего окна (страницы) в соответствии с названием файла для этой страницы
  * Производимые действия
    - Вернуть значение переменной current_page класса App
