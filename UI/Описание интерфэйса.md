# Описание файлов с UI
  ## sign_up_page
  * Страница с UI, который предназначен для осуществления регистрации и входа в аккаунт
    ### Элементы
      - sign_up_page: рамка в которую помещаются элементы страницы
      - name_entry и username_entry: поля для ввода имени и имени пользователя (имя убирается при входе в аккаунт)
      - submit_button: кнопка для подтверждения введённых данных
      - login_button: кнопка для переключения на вход в аккаунт
      - error_label: надпись для отображения ошибок
    ### Параметры
      - master: главное окно ткинтер (Tk)
      - register_method: метод для регистрации
      - login_method: метод для входа
    ### Методы
      - show_sign_up_page
      - hide_sign_up_page
  ## chatting_page (Главное окно)
  * Страница с UI, который предначначен для выбора чата и для непосредственной переписки с другими пользователями
    ### Элементы
      - chats_list_frame: рамка для списка чатов
      - add_new_chat_button: кнопка для перехода на страницу для добавления нового чата
      - chat_frame: рамка для выбранного показа чата и сообщений в нём
    ### Параметры
      - master
      - switch_to_adding_page: метод для перехода на страницу для добавления чата
    ### Методы
      - show_chatting_page
      - hide_chatting_page
  ## start_page
  * Начальная страница, появляется при запуске на 2 секунды затем исчезает, можно разместить логотип
    ### Элементы
      - start_page_frame: рамка для размещения элементов
      - welcome_label: (временно) привествие пользователя
    ### Параметры
      - master
    ### Методы
      - show_start_page
      - hide_start_page
